Hi! This website was made as a portfolio project, to learn Angular and practice with Typescript. It also uses some Material Design buttons/icons/switches, and TailwindCSS for styling.

It shows rain data and traffic data worldwide, and train disruptions for the Dutch railways, all at once. (I'll admit it's a bit random, thinking of practice projects can be a challengeðŸ™ƒ)

How It's Made
The map is based on Leaflet.js. For the data (Leaflet tiles), I used:

  • the NS API for railway disruptions, which provides GeoJSON data, allowing for custom colors, labels, popups, etc,
  • the TOMTOM API for traffic data, which gives data in PNG format (per tile, depending om zoom/location),
  • and the Rainviewer plugin for leaflet which I have no clue how it works exactly and where it gets the data from, but it works ðŸ‘Œ.


The API's are all free versions, so they might run out of requests quickly after which those parts will stop working (you should get a warning!) :)
